# How to Upload Files to GitHub (No Git Required)

## Step-by-Step Instructions

### 1. Go to Your GitHub Repository
- Open your web browser
- Go to https://github.com
- Sign in to your account
- Navigate to your repository (the one connected to Render)

### 2. Upload Files
- Click the **"Add file"** button (top right of the file list)
- Select **"Upload files"** from the dropdown

### 3. Add Your Files
- Drag and drop these files into the upload area:
  - `pricing.html`
  - `monday_logo.png`
- OR click "choose your files" and browse to select them

### 4. Commit Changes
- Scroll down to the bottom of the page
- In the "Commit changes" section:
  - Add a commit message: "Update logo and remove Free text"
  - Leave "Commit directly to the main branch" selected
- Click the green **"Commit changes"** button

### 5. Wait for Render to Deploy
- Render will automatically detect the changes
- Wait 2-3 minutes for the deployment to complete
- Your site will update at: https://aipricing26.onrender.com/
