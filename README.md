# monday.com × Anthropic Partnership Deck

A presentation deck showcasing the partnership initiatives between monday.com and Anthropic.

## Viewing Locally

Simply open `brand-deck.html` in your web browser.

## Publishing & Sharing

### Option 1: Netlify Drop (Easiest - No Account Required)

1. Go to [https://app.netlify.com/drop](https://app.netlify.com/drop)
2. Drag and drop the `brand-deck.html` file onto the page
3. You'll get an instant shareable URL (e.g., `https://random-name-123.netlify.app`)
4. Share this URL with anyone!

### Option 2: GitHub Pages (Free, Permanent)

1. Create a GitHub account at [github.com](https://github.com)
2. Create a new repository
3. Upload `brand-deck.html` to the repository
4. Go to Settings → Pages
5. Select the main branch and save
6. Your deck will be available at `https://yourusername.github.io/repository-name/brand-deck.html`

### Option 3: Vercel (Free, Fast)

1. Go to [vercel.com](https://vercel.com) and sign up
2. Click "New Project"
3. Drag and drop the `brand-deck.html` file
4. Get an instant shareable URL

### Option 4: Simple File Hosting

You can also upload the HTML file to any web hosting service:
- Google Drive (make file public and share link)
- Dropbox (create public link)
- Any web hosting service you have access to
