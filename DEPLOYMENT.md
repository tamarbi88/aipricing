# How to Host Your Pricing Page

## Option 1: Netlify Drop (Easiest - No Account Needed)

1. Go to https://app.netlify.com/drop
2. Drag and drop your entire project folder (`New Project`) onto the page
3. You'll get an instant URL like `https://random-name-123.netlify.app`
4. Share this URL publicly

**Note:** Make sure all your image files are in the same folder as `pricing.html`

## Option 2: GitHub Pages (Free, Permanent)

1. Create a GitHub account at https://github.com
2. Create a new repository
3. Upload all your files (pricing.html + images)
4. Go to Settings > Pages
5. Select your main branch and save
6. Your site will be available at `https://yourusername.github.io/repository-name/pricing.html`

## Option 3: Vercel (Free, Fast)

1. Go to https://vercel.com
2. Sign up with GitHub (or email)
3. Click "Add New Project"
4. Drag and drop your project folder
5. Get an instant URL

## Quick Fix: Rename for Easy Access

If using GitHub Pages or similar, you might want to rename `pricing.html` to `index.html` so it loads automatically at the root URL.
